HeadRush

Adobe AIR online multiplayer open-source game

Download as .zip Download as .tar.gz View on GitHub

Sup. This is Head Rush. It's an online multiplayer 2D platformer shooter. In other words, you move like Mario, you shoot like Master Chief, and all this is online.

It's gonna blow your mind. 10/10 -Me

It's built using Streamy and Flixel.
